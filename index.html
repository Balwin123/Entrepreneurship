Most important Git commands are:

  1. Git desktop: to navigate to desktop
  2. Git mkdir: create a new folder
  3. Git touch: create a new file
  4. Code . +enter: open a code editor

Most important git commands to handle github:

  • Git clone URL: to clone empty repository
  • Git add . +enter: Add all files & folders to stage area
  • Git commit -m "message"
  • Git push: It will push all codes to girhub

How to Deploy website on gitHub:

  • Click on settings
  • Scrol down until you see GitHub pages
  • Click on branch "none" and select "master" from dropdown
  • Click on Save
  • Copy the link and share with freinds & familys
